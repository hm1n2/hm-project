# AGENTS.md

이 파일은 이 프로젝트의 공통 작업 지침 원본이다.

## 작업 방식

- 이 프로젝트는 기획 먼저 하는 방식으로 진행한다.
- 코드부터 작성하지 않는다.
- 먼저 `docs/`에서 기획을 끝낸 뒤 구현으로 넘어간다.

## 커뮤니케이션

- 모든 대화와 설명은 한국어로 진행한다.
- 확실하지 않으면 먼저 질문하고 진행한다.

## 파일 변경

- 파일을 삭제하기 전에는 반드시 먼저 확인을 받고 진행한다.

## Git

- `git` 커밋은 의미 있는 단위로 나눈다.
- 커밋 메시지는 한국어로 작성해도 된다.
