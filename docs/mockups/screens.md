# Mockup Screens — TRADDY

## Overview

각 화면의 정보 구조·레이아웃·상태·리뷰 질문을 상세화한 문서다. `plan.md`의 화면 ID와 완전히 일치한다. ASCII 도식은 화면의 정보 구조를 빠르게 이해하기 위한 것으로 실제 구현 와이어프레임이 아니다.

---

## Global Mockup Assumptions

- **레이아웃:** 단일 컬럼, max-width 420px, 상단 헤더 56px 고정, 하단 내비게이션 52px + safe-area
- **색상:** 배경 #ffffff, Primary CTA #eb2b51, 민트 강조 #1ec7be, 텍스트 #323232
- **폰트:** Pretendard, Body 15px/400, Button 16px/600, Caption 13px/400
- **더미 데이터:** 사용자 이름(지수·민호·현아), 지역(파리 마레·오르세), 시간대(오늘 저녁·내일 오전)
- **반응형:** 모바일 100% 너비, 데스크톱 420px 센터 정렬 고정
- **아이콘:** lucide-react, 24px standalone / 20px button
- **Drawer/Modal:** vaul-drawer bottom-sheet 패턴, scrim rgba(0,0,0,0.1)
- **이모지 없음:** 상태 표시는 아이콘 또는 컬러 도트로 처리
