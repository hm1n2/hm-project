## 확정 정보
- **서비스 이름:** TRADDY
- **한 줄 무드:** 낯선 도시에서도 안심을 먼저 건네고, 함께할 가능성을 설레게 열어주는 든든한 여행 메이트

## 1. Voice Summary
Sounds like 낯선 도시에서 먼저 긴장을 풀어주고, 혼자만의 여행도 존중하면서 새로운 사람과의 하루를 부드럽게 연결해주는 여행 메이트 who 안심과 설렘 사이의 균형을 잃지 않고 사람을 이어준다.

## 2. Core Personality Traits
**설레는**
- We are: 새로운 사람과의 만남이 여행을 조금 더 특별하게 만들 수 있다고 자연스럽게 제안한다.
- We are not: 가볍고 공허한 분위기로 감정만 부풀리지는 않는다.
- This sounds like: "오늘 저녁이 꼭 혼자여야 하는 건 아니에요."
- This does NOT sound like: "지금 안 움직이면 좋은 사람 다 놓쳐요."

**친근한**
- We are: 처음 쓰는 사람도 부담 없이 이해할 수 있도록 부드럽고 편하게 말한다.
- We are not: 오래 안 사이처럼 선을 넘거나 사적인 친밀감을 강요하지 않는다.
- This sounds like: "비슷한 결의 사람을 천천히 찾아볼까요?"
- This does NOT sound like: "오늘은 누구랑 썸 타러 나가볼까요?"

**센스 있는**
- We are: 사용자의 상황을 눈치 있게 읽고, 다음 행동을 자연스럽게 이끌어준다.
- We are not: 규칙을 빡빡하게 들이밀거나 잘난 척하는 말투를 쓰지 않는다.
- This sounds like: "조건을 조금만 넓혀보면 더 잘 맞는 사람이 보일 수 있어요."
- This does NOT sound like: "설정이 비효율적이에요. 다시 맞춰주세요."

**즉흥적인**
- We are: 오늘의 동선과 기분에 맞춰 가볍게 연결되는 여행의 재미를 살린다.
- We are not: 믿음이 떨어질 정도로 가볍거나 무책임하게 굴지 않는다.
- This sounds like: "가고 싶은 곳이 있다면, 같은 결의 사람과 함께일 수도 있어요."
- This does NOT sound like: "일단 만나보고 아니면 말죠."

## 4. Vocabulary Guide
**USE**
- 같은 결: 여행 취향이 잘 맞는 느낌을 부드럽게 표현할 때
- 가볍게: 부담 없이 시작할 수 있다는 인상을 줄 때
- 천천히: 조급함 없이 탐색해도 된다는 안정감을 줄 때
- 안심: 낯선 사람과의 만남에 대한 불안을 덜어줄 때
- 함께일 수도 있어요: 가능성을 제안하되 강요하지 않을 때
- 잘 맞는 사람: 매칭의 질을 강조할 때
- 오늘의 동행: 당일성, 즉흥성을 담아낼 때
- 비슷한 루트: 현재 일정 기반 탐색을 쉽게 설명할 때
- 먼저 둘러보기: 바로 행동을 강요하지 않고 진입 장벽을 낮출 때
- 분위기: 식당, 장소, 무드를 감각적으로 설명할 때
- 자연스럽게: 억지스럽지 않은 연결감을 줄 때
- 든든하게: 안전감과 신뢰를 강조할 때

**AVOID**
- 놓쳐요: 조급함과 압박을 주는 표현이라 금지
- 지금 당장: 명령형 독촉처럼 들릴 수 있어 금지
- 벌금/끊겨요/사라집니다: 불안을 자극하는 서비스처럼 보일 수 있어 금지
- 썸/설렘 폭발: 소개팅·헌팅앱처럼 읽힐 위험이 있어 금지
- 매력적인 사람: 외모·로맨스 중심으로 오해될 수 있어 금지
- 분발하세요: 평가하거나 다그치는 느낌이라 금지
- 안전을 위해 반드시: 보호자처럼 과잉 통제하는 인상을 줄 수 있어 주의
- 완벽한 동행: 과장되고 공허하게 들릴 수 있어 금지
- 힙한 영어 남발: 처음 쓰는 사용자에게 거리감을 줄 수 있어 금지
- 무조건: 선택의 여지를 지우는 강한 단정이라 금지
- 빨리: 속도를 강요하는 느낌이라 금지
- 소개팅 같은 비유: 서비스 포지셔닝을 흐릴 수 있어 금지

## 5. Tone Adaptation Rules
**By Context (앱 안 맥락):**

| 맥락 | 톤 조절 | 예시 문구 |
|------|---------|-----------|
| 온보딩 / 첫 화면 | 설렘과 친근함을 조금 더 키우고, 설명은 짧고 부드럽게 제안형으로 쓴다. | "가고 싶은 곳이 있다면, 같은 결의 사람과 함께일 수도 있어요. 오늘의 동행을 가볍게 찾아볼까요?" |
| 빈 상태(empty state) | 실망감을 줄이지 않도록 안심을 먼저 주고, 다음 행동을 센스 있게 이끈다. | "아직 딱 맞는 동행은 안 보이네요. 대신, 비슷한 루트의 사람들부터 가볍게 둘러볼 수 있어요." |
| 성공/완료 피드백 | 과한 축하보다 기분 좋은 확신을 주고, 다음 기대를 살짝 남긴다. | "좋아요. 오늘의 동행을 한 걸음 더 가까이 찾았어요 ✨" |
| 에러/안내 | 책임을 사용자에게 돌리지 않고, 차분하고 명확하게 다시 시도할 길을 보여준다. | "잠시 연결이 흔들렸어요. 한 번만 다시 불러올게요." |
| 푸시 알림 | 독촉하지 않고 가능성을 상기시키는 정도로만 가볍게 건넨다. | "근처에 비슷한 결의 여행자가 보이고 있어요. 원할 때 가볍게 확인해보세요." |
| 랜딩/소개 페이지 | 혼자 여행을 존중하면서도 함께의 즐거움을 여는 브랜드 메시지를 분명히 드러낸다. | "혼자 떠난 여행도, 누군가와 함께라면 또 다른 하루가 될 수 있어요." |

## 6. Style Rules
- 존댓말 기본: 부드러운 존댓말을 사용한다. ("~해요", "~볼까요?")
- 문장 길이: 짧고 쉽게 읽히는 문장을 우선한다.
- 제안형 표현: 명령형보다 제안형을 쓴다. ("찾아볼까요", "둘러볼 수 있어요")
- 이모지: 축하나 반가움이 어울리는 맥락에서만 최대 1개 사용한다.
- 외래어: 트렌드에 맞는 범위에서 자연스럽게 쓰되, 이해를 방해할 만큼 남발하지 않는다.
- 느낌표: 꼭 필요한 감정 강조에만 제한적으로 사용한다.
- 불안 자극 금지: 위험, 놓침, 실패를 과장해 행동을 유도하지 않는다.
- 관계선 유지: 지나치게 사적인 친밀감이나 flirting처럼 읽히는 표현은 피한다.

## 7. Do's and Don'ts
**Do**
- 혼자 여행하는 선택 자체를 존중하는 문장을 쓴다.
- 함께할 가능성을 부드럽게 제안하는 표현을 쓴다.
- 처음 쓰는 사람도 바로 이해할 수 있게 쉽게 설명한다.
- 설렘을 주더라도 신뢰와 안심을 바탕에 깐다.
- 빈 상태나 실패 상황에서도 사용자를 다그치지 않는다.
- 사용자의 오늘 일정과 분위기를 읽어주는 듯한 문장을 쓴다.

**Don't**
- 조급함을 유도하는 독촉형 문장을 쓰지 않는다.
- 소개팅앱이나 헌팅앱처럼 읽히는 표현을 쓰지 않는다.
- 안전만 지나치게 강조해 여행의 재미를 눌러버리지 않는다.
- 즉흥성을 이유로 무책임하거나 가벼운 분위기를 만들지 않는다.
- 사용자를 평가하거나 훈계하는 말투를 쓰지 않는다.
- 과장된 약속이나 공허한 감성 문구로 신뢰를 깎지 않는다.
